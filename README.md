Delicious dotfiles
=====

Don't forget to install the following packages before you install these dots:

Basic packages
-----

- zsh and grml-zsh-config
- cower (aur)
- htop
- ponymix (aur)
- ranger
- tmux
- vim
- mpd
- ncmpcpp
- mpdscribble

X-dependent packages
-----

- i3 and/or openbox
- conky
- compton
- dmenu
- dunst
- feh
- xfce4-volumed-pulse (aur)

Openbox-oriented packages
-----

- neap (aur)
- obmenu-generator (aur)
- tint2
- skippy-xd (aur)
